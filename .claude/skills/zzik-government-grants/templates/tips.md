## 1. 기술 혁신성 (Technology Innovation)

### 1.1 핵심 기술 개요

**기술명**: LLM-Powered Location-based Stablecoin Payment Platform

**한줄 요약**:
LLM 슈퍼앱 시대의 위치기반 스테이블코인 결제 인프라. ChatGPT와 같은 AI 에이전트가 자동으로 장소를 추천하고 USDT/USDC로 결제할 수 있는 플랫폼.

### 1.2 기술의 독창성 (Novelty)

#### 기존 기술의 한계

**Google Maps + Stripe 모델**:
- ❌ 한국 내비게이션 제한
- ❌ 신용카드 결제만 지원
- ❌ 환전 수수료 3-5%
- ❌ LLM 슈퍼앱 연동 없음

**네이버/카카오 지도**:
- ❌ 한국어만 지원
- ❌ 외국인 UX 미흡
- ❌ Web3 결제 미지원

#### ZZIK의 혁신

**1. LLM Agentic Commerce 최초 구현**
```
User → ChatGPT: "성수동 힙한 카페 추천해줘"
ChatGPT → ZZIK API: 장소 검색 (스테이블코인 가능 매장)
ZZIK → ChatGPT: 테라로사 추천 (USDT 결제 가능)
ChatGPT → User: "테라로사 추천합니다. ZZIK으로 USDT 결제하세요"
User → ZZIK: One-Click Payment (MetaMask)
```

**2. 이중 법인 구조로 규제 준수**
```
[ZZIK Korea] → 한국인 대상 (KRW 결제)
[ZZIK Global] → 외국인 대상 (USDT/USDC 결제)
→ AIFC VASP 라이선스로 합법적 운영
```

**3. GPS 정확도 99.9% (실내 포함)**
- 7-Layer Anti-Spoofing (속도 추적, IP 교차 검증)
- QR 코드 이중 검증 (HMAC + Nonce)
- Chainalysis FDS (자금세탁방지)

| 기술 요소 | 기존 기술 | ZZIK 혁신 | 우위 |

## 6. 기술 상세 (Technical Details)

### 6.1 Architecture

```
[User] → [ChatGPT / HyperCLOVA X]
          ↓ (Function Calling)
       [ZZIK API]
          ├─ /api/llm/places (장소 검색)
          ├─ /api/llm/payment (결제 세션 생성)
          └─ /api/llm/verify (트랜잭션 검증)
          ↓
       [PostgreSQL] (매장 데이터)
       [Polygon Network] (USDT/USDC)
       [Chainalysis API] (FDS)
```

### 6.2 Tech Stack

**Frontend**:
- Next.js 14.2 (App Router)
- React 19
- ethers.js (Web3 integration)
- PWA

**Backend**:
- Node.js + PostgreSQL 17
- Prisma 7 ORM
- Redis (Upstash)

**Blockchain**:
- Polygon Network (저렴한 가스비)
- USDT/USDC (ERC-20 tokens)
- Chainalysis KYT (Know Your Transaction)

**AI**:
- OpenAI GPT-4 (Agentic Commerce)
- Naver HyperCLOVA X (제휴 예정)
- Google Translate API (42개 언어)

### 6.3 Security

**7-Layer GPS Anti-Spoofing**:
1. 속도 추적 (velocity check)
2. IP 위치 교차 검증
3. Device fingerprinting
4. 시간 패턴 분석
5. QR 코드 이중 검증
6. Chainalysis FDS
7. 수동 리뷰 (의심 거래)
