Applied behavioral science for ZZIK MAP:

**Part 1: Cognitive Biases (6 major biases)**
- Anchoring Bias: "847 travelers" sets credibility anchor
- Loss Aversion: "Save your journey" appeals to fear of loss
- Status Quo Bias: Encourage trying new vibes/features
- Choice Paradox: 8 vibe categories (optimal, not paralyzing)
- Recency Bias: Show trends over multiple timeframes

- Daily motivation flow from curiosity → autonomy → mastery → social → purpose

**Part 3: Habit Formation Loop**

**Part 4: Mental Models & Metaphors**
- Journey Intelligence: "Follow the crowd wisdom"
- Vibe Matching: "Match the feeling, not the place"
- MAP BOX: "Collect experiences like a scrapbook"

**Part 5: Decision-Making Framework**
- 5-stage user decision journey with ZZIK optimization at each stage

**Part 6: Error Recovery & Resilience**
- Strategies for each error type (upload fails, GPS fails, no results)

**Part 7: Trust Building**
- Initial trust (first 10 seconds) → Sustained trust (first week) → Deep trust (long-term)

#### 1. **motion-economics-v7.md** (616 lines)
**Location**: `.claude/skills/motion-economics-v7.md`

Animation ROI & performance psychology:

**Part 1: Motion ROI Framework**
- Cost-benefit analysis for every animation
- ROI formula: (cognitive_value + engagement_value) / (performance_cost + memory_cost)
- ZZIK animation analysis (high/medium/low ROI animations)

**Part 2: Animation Performance Thresholds**
- FPS targets: 55-60 fps (smooth), < 40 fps (jank)
- Animation budget: 200KB total code
- Frame time: ≤ 16.67ms per frame

**Part 3: Animation Psychology**
- Duration psychology: 50ms → instant, 100-200ms → responsive, 300-500ms → intentional
- Easing curves: easeOut (responsive), easeIn (deliberate), spring (celebratory)

**Part 4: Motion & Accessibility**
- prefers-reduced-motion support
- Safe animation wrapper patterns

**Part 5: Motion Hierarchy**
- Tier 1 (Always): Button feedback, form validation, loading states
- Tier 2 (High-Value): Page transitions, celebrations, modals
- Tier 3 (Nice-to-Have): List stagger, hover reveals, parallax
- Tier 4 (Remove): Distracting background animations

**Part 6: Motion Metrics**
- Perceived responsiveness, speed, delight
- FPS, paint time, layout shift, TTI targets

**Part 7: Optimization Checklist**
- 14-point checklist before shipping any animation
