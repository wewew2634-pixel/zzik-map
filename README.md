# ZZIK MAP

> AI-Powered K-Travel Discovery Platform

**"With one photo, know where similar travelers went next"**

## Overview

ZZIK MAP helps foreign tourists discover their next Korean destination by analyzing travel patterns from similar travelers.

## Features

- **Journey Intelligence**: Photo-based next destination recommendations
- **Vibe Matching**: Find places with similar atmosphere
- **MAP BOX**: Creator-business experience marketplace

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL + pgvector)
- **AI**: Gemini 2.0 Flash Vision
- **Maps**: Kakao Maps, Mapbox

## Getting Started

```bash
cd app
pnpm install
pnpm dev
```
